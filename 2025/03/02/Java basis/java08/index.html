Java类的基础知识

类

类表示方法及处理的数据对象的集合的结构，比方法更大一级的控件。

封装：将字段设为私有，防止来自外部的访问，从而使方法和字段紧密结合在一起的操作

getter和setter方法统称为访问器：以get开头的方法用于获取、返回字段的值；以set开头的方法用于设置字段的值

注意：在该类中创建了Day类型的开户日期，存款时间和存款结束时间

构造函数的重载是为了后续创建类类型变量时new Day()中()内的参数可以有0个、1个、2个…全部，这样比较方便

类（实例）变量和类（实例）方法

  • 类变量(静态变量):它是所有实例共享的数据，与实例无关，且只有一个，要加上static，不加即为实例变量
  • 实例初始化器{…}：每次创建实例前都会执行实例初始化器其中不能存在return语句，可以存在this语句和super语句；
  • 类初始化器static{…}，只执行一次，其中不能存在return语句、this语句和super语句。

  • Math类调用方法（不是实例方法）[类名.静态方法名]：Math.sqrt()、Math.max(x,y)、Math.min(x,y)、 返回x的绝对值Math.abs(x)
