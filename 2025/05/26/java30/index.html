Java File与IO流

变量、数组、对象、集合都是存储数据的容器，但是当在断电或者程序终止时会丢失。要想将数据长久保存起来，会使用文件，文件是在计算机硬盘中，这样数据不会在之前那种情况下而丢失。

File类只能对文件本身操作，不能读写文件中的数据，而IO流可以读写文件或网络中的数据

File

File是java.io包下的类，File类的对象，用于代表当前操作系统的文件（文件、文件夹）

常用方法

获取文件信息

  • public boolean exists()判断文件路径是否存在
  • public boolean isFile()判断是否为文件
  • public boolean isDirectory()判断是否为文件夹
  • public long length()获取文件的字节长度
  • public String getName()获取文件名字
  • public long lastModified() 获取文件的最后修改时间(时间毫秒值)
  • public String getPath()获取创建文件对象时使用的路径,new File的路径
  • public String getAbsolutePath()获取绝对路径

创建删除文件

  • public boolean createNewFile()创建文件，内容为空
  • public boolean mkdir()创建一级文件夹
  • public boolean mkdirs()创建多级文件夹
  • public boolean delete()删除文件或空文件夹，不可以删除非空文件夹


遍历文件夹一级文件

  • public String[] List()获取当前目录下所有的“一级文件名称”到一个字符串数组中返回
  • public File[] listFiles()获取当前目录下所有的“一级文件对象”到一个文件对象数组中返回


文件夹为空或者路径不存在则返回null，空文件夹则返回空数组，隐藏文件夹也可被遍历出来

方法调用自身的形式称为方法递归。递归的形式：直接递归（方法自己调用自己）与间接递归（方法调用其他方法，其他方法调用自己）

案例——计算n的阶乘

计算n的阶乘，5的阶乘=12345
所以f(n)=1234…(n-1)*n——>*f(n)=f(n-1)n

递归算法三要素：

  • 递归公式：f(n)=f(n-1)*n
  • 递归的终结点：f(1)
  • 递归的方向必须走向终结点

经典面试题——猴子吃桃

第一天猴子摘了若干个桃子，吃了一半不过瘾，又吃了一个，第二天又吃了前一天剩余桃子的一半，又觉得不过瘾，又多吃了一个，此后每天都吃前一天剩余桃子的一半加一个桃子，第十天发现只有1个桃子了，请问猴子第一天摘了多少个桃子？

分析：

  • f(10)=1;
  • 第一天有f(1)个桃子
  • 第2天有f(2)=f(1)-f(1)/2-1桃子
  • 第n天有f(n)=f(n-1)-f(n-1)/2-1
  • 令n=n+1，f(n+1)=f(n)-f(n)/2-1;
  • f(n)=2*f(n+1)+2
