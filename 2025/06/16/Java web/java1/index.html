Java web前端 HTML、CSS

Web

  • Web是全球广域网，也称万维网（WWW：World Wide Web），能够通过浏览器访问的网站
  • Web由三部分组成：前端程序+服务端程序+数据库

网页由三个部分组成：

  • HTML:负责网页的结构（页面元素和内容）
  • CSS：负责网页的表现（元素的外观、位置等页面样式，如颜色、大小等）
  • JavaScipt：负责网页的行为（交互效果）

HTML

HTML称为超文本语言

  • 超文本：超越文本限制，比普通文本更强大，除了文字信息，还可以定义图片、音频、视频等内容
  • 标记语言：由标签”<标签名>”构成的语言
    • HTML标签都是预先定好的，例如<h1>展示标题，使用<img>展示图片等
    • HTML代码直接在浏览器中执行，HTML标签由浏览器解析

HTML标签

  • 超链接：<a href=""></a>

  • 行内标签:<span></span>

  • 视频标签：<video src="视频地址"></video>

  • 换行：<br>

  • 音频标签：<audio src="音频地址"></audio>

  • 段落标签：<p></p>

  • 图片(动图)标签：<img src="图片地址"></img>

  • 加粗标签：<b></b>、<strong></strong>(带有强调意味)

  • 下划线标签：<u></u>、<ins></ins>(带有强调意味)

  • 斜体标签：<i></i>、<em></em>(带有强调意味)

  • 删除线标签：<s></s>、<del></del>(带有强调意味)

实体字符

  • 空格：&nbsp;
  • <:&lt;
  • >:&gt;

CSS

CSS是层叠样式表，用于控制页面的样式

CSS的用法：在style中设置样式

样式

  • text-decoration: none;去除超链接下方的下划线

添加CSS样式的三种方法

  • 行内样式：在标签中添加style属性，属性值是样式

选取器

选择器是CSS最基本的组成单元，选择器用于选择HTML元素，然后应用CSS样式。

  • 元素选择器：元素名称{...},例如h1{...}，选择页面上所有的<h1></h1>标签。
  • id选择器:#id属性值{...},例如：#hid{...}，选择id属性值为hid的标签。
  • class选择器:.class属性值{...},例如：.cls{...}，选择页面上所有的class属性为cls的标签，唯一的标识。
  • 属性选择器:
    • 元素名称[属性名="属性值"]{...},例如：input[type="hid"]{...}，选择页面上type属性为hid的<input>标签。
    • 元素名称[属性]{...},例如：input[type]{...}，选择页面上type属性为text的<input>标签。
  • 分组选择器:.class属性值1,.class属性值2{...},例如：.cls1,.cls2{...}，选择class属性值为cls1或cls2的标签。
  • 后代选择器：元素1 元素2{...},例如：form input{...}，选择form标签内所有的input标签。

优先级：id选择器>class选择器>元素选择器

视频标签样式

  • src: 视频地址
  • controls：显示播放控件
  • autoplay: 自动播放
  • width：视频宽度（像素px、百分比%）
  • height：视频高度（像素px、百分比%），宽度高度只设置一个，另一个会等比例缩放

图片标签样式

  • src：图片路径
    • 绝对磁盘路径：C:\Users\hhd0717\Pictures\huang.jpg
    • 绝对网络路径：http://www.baidu.com/huang.jpg
    • 相对路径：
      • ./：当前目录（可以省略），比如./huang.jpg
      • ../：上一级目录，比如../huang.jpg
  • width：图片宽度
  • height：图片高度，（像素px、百分比%），宽度高度只设置一个，另一个会等比例缩放

行高和缩进

  • 设置段落的行高：line-height: 2;，两倍行高
  • 设置段落缩进：text-indent: 2em;，缩进两字符
