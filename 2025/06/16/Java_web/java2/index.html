Java web前端 JavaScript

**JavaScript（JS）**是一门跨平台、面向对象的脚本语言（无需编译），用来控制王爷行为，实现页面的交互效果。

组成

  • ECMAScript 规定了JS基础语法核心知识，包括变量、数据类型型、运算符、控制结构、函数、对象等。
  • BOM：浏览器对象模型，通过浏览器本身，如页面弹窗、地址栏操作、关闭窗口等。
  • DOM：文档对象模型，通过操作HTML文档，如获取元素、修改元素、添加元素等。

核心语法

引入方式

  • 内部脚本：将JS代码定义在HTML文档中

    • 放在<script></script>标签内。
    • 可以放在HTML的任意位置
    • 建议会把脚本置于<body>元素的底部，可改善显示速度

  • 外部脚本：将JS代码保存在单独的JS文件中，然后引入到HTML页面中<script src="xxx.js"></script>
    在js包下新建文件demo.js,写入alert("hello world!");
    在HTML页面中引入demo.js文件<script src="js/demo.js"></script>,最终达到同样的效果

书写规范：每行结尾以分号结尾

基础语法

变量

  • 在JS中，使用let关键字s声明变量，（弱类型语言），变量可以存放不同类型的值
  • 变量名必须以字母、下划线、$开头，不能以数字开头，变量名区分大小写。
  • 变量名不能使用关键字，如let、var、if、for。

常量

  • 在JS中，使用const关键字声明常量，常量一旦声明，不能修改它的值（不能重新赋值）。
