Java泛型

利用ArrayList集合认识泛型

  • 不限制元素类型（支持泛型）：ArrayList list = new ArrayList();
  • 支持一种非基本类型的类型：ArrayList<String> list = new ArrayList<>();
  • 支持一种类类型：ArrayList<Student> list = new ArrayList<>();
