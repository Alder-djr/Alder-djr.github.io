JavaAPI(一)——Object(s)、包装类、StringBuilder/StringBuffer/StringJoiner

Java API是java提供给我们的程序，方便我们完成功能，其构成也是多个类组成的包

Object类

Object类是所有类的祖宗类,所以Java中的所有类的对象都可以直接使用Object类中提供的一些方法

  • equals()
    • 基本作用：判断两个对象的地址是否相同，与==相同。
    • 所以一般情况下需要重写，用来判断两个对象内容是否相同
  • toString()
    • 基本作用：返回对象的字符串形式
    • 让子类重写，以便返回子类对象的内容
  • clone()
    • 基本作用：克隆一个相同的对象

浅克隆与深克隆

  • 浅克隆：基本类型对象直接拷贝内容，引用类型对象（如String）拷贝的只是地址
  • 深克隆：基本类型对象直接拷贝内容，引用类型对象（如String）拷贝的只是地址，其他对象（如数组）不会拷贝地址，而是会创建新对象

  • isNull():判断对象是否为空，为空则为true

包装类

包装类是把基本数据类型的数据包装成对象

基本数据类型 对应的包装类（引用数据类型）
byte Byte
short Short
int Integer
long Long
char Character
float Float
double Double
boolean Boolean
  • 将基本数据类型转换为包装类型Integer a=Integer.valueOf(12)
  • 自动装箱，基本数据类型自动转换为包装类型Integer b=12
  • 自动拆箱,可以自动把包装类型的对象转化为对应的基本类型数据int a4=a;自动拆箱主要使用在集合中
  • 把基本类型或包装类型数据转换为字符串toString()
  • 把字符串类型的数值转换成对应的基本类型Integer.valueOf
    • 一般在网页中用户输入字符串类型，但在后期需要转换城基本类型。

StringBuilder与StringBuffer

StringBuilder

StringBuilder表示可变字符串对象，相当于是一个容器，它里面装的字符串是可以改变的，就是用来操作字符串的。

  • （有参无参）构造器StringBuilder()、StringBuilder(String str)
  • 拼接字符串：append()
  • 支持链式编程：append().append()...
  • 反转操作：reverse()
  • 返回字符串长度：length()
  • 将StringBuilder对象转换成String对象：toString()

StringBuffer

StringBuffer与StringBuilder用法是相同的，但是StringBuilder相比于StringBuilder线程更加安全；当我做一个系统需要多人访问StringBuilder对象，这是建议改成StringBuffer更加安全。
